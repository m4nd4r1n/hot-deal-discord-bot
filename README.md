<br>

## 기능

설정한 카테고리의 새 글을 전송합니다.

<br>

## 명령어

`!help`, `!h`, `!명령어`, `!도움말` - 명령어 목록을 출력합니다.

`!알림` - 명령어를 입력한 채널로 알림을 전송합니다.

`!해제` - 명령어를 입력한 채널의 알림을 해제합니다.

`!카테고리` - 현재 알림 설정된 카테고리를 보여줍니다.

`!카테고리 추가` - 알림 받을 카테고리를 추가합니다.

`!카테고리 제거` - 알림 설정한 카테고리를 제거합니다.
